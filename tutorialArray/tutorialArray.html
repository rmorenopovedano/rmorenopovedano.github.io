      2. Unshift(): Añade elementos al principio del array.

    3. ¿Devuelven algún valor?
      4. Shift(): devuelve el valor del primer elemento del array.

    5. Si buscas añadir elementos al final, el método push(). Si quieres añadir elementos al inicio, el método unshift(). En el caso de querer eliminar elementos, pop() en caso de querer hacerlo del final, y shift() en caso de querer eliminar elementos del principio del array. Pero si tu objetivo es añadir o quitar elementos del array sea donde sea, es mejor usar pop() y push(), ya que al quitar elementos del final, es más fácil de reorganizar el array que si usamos el shift() o unshift(), ya que añade y elimina del principio y realizaría más procesos para reorganizar el array.
  5. join() y split(), ¿son métodos de String o de Array?
    6. Join() es un método de Array para convertir en String. Mientras que split() es un método de Strings que los convierte en Array.
  6. splice()
    1. Utilidad
      2. splice(): Cambia el contenido de un array elimando elementos existentes o añadiendo nuevos elementos.
      Sintaxis: array.splice(start, deleteCount[, item1[, item2[, ...]]])
    2. Indica significado del primer argumento admitido: índice, cantidad o contenido
      3. El primer argumento es el índice. Es el lugar desde donde va a comenzar a eliminar elementos del array.
    3. Indica significado del segundo argumento admitido: índice, cantidad o contenido
      4. El segundo argumento es la cantidad de elementos que quieres eliminar del array. Si este parámentro es 0 no se elimina ningún elemento del array, y si este valor es mayor que el número de elementos que contiene el array, se borrarán todos.
    4. Indica significado del resto de argumentos admitidos: índice, cantidad o contenido
      5. El resto de argumentos indican el valor del elemento que se quiere añadir al array. Si no se especifica este argumento, splice() solo eliminará elementos, sin añadir ninguno.
    5. ¿Modifica el array original?
      6. Con el método splice() se modifica el array original.
  7. slice()
    1. Utilidad
      2. Selecciona una parte del array y devuelve un array de con la parte seleccionada.
      Sintaxis: arr.slice([begin[, end]])
    2. Indica significado del primer argumento admitido: índice, cantidad o contenido
      3. El primer argumento es el índice inicial. Es el lugar desde donde inicia a seleccionar los elementos que queremos extraer.
    3. Indica significado del segundo argumento admitido: índice, cantidad o contenido
      4. El primer argumento es el índice final. Es el lugar donde acaba la selecciónde los elementos que queremos extraer.
    4. ¿Modifica el array original?
      5. Con el método slice() no se modifica el array original. sino que copia un pedazo de él.
